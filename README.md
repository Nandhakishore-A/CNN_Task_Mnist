# CNN_Task_Mnist

CNN Handwritten Digit Recognition using MNIST

A Deep Learning project that uses a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset using TensorFlow and Keras.

📌 Project Overview

This project demonstrates how CNNs can be used for image classification tasks.
The model is trained on the MNIST dataset, which contains handwritten digits from 0 to 9.

The project includes:

Data preprocessing
CNN model creation
Model training
Prediction
Accuracy evaluation
Visualization using Matplotlib

🚀 Technologies Used
Python
TensorFlow
Keras
NumPy
Matplotlib
Scikit-learn

📂 Dataset

The project uses the built-in MNIST dataset provided by Keras.

Dataset contains:

60,000 training images
10,000 testing images
Image size: 28 × 28 pixels

🧠 CNN Architecture

The CNN model consists of:

Convolution Layer
ReLU Activation Function
Max Pooling Layer
Second Convolution Layer
Flatten Layer
Dense Hidden Layer
Output Layer with Softmax Activation

⚙️ Features
Image normalization
CNN-based image classification
Handwritten digit prediction
Accuracy and loss visualization
Confusion matrix generation

📊 Model Performance
Achieved high accuracy on the MNIST test dataset
Efficient digit classification using CNN architecture
